## Description

ITMO.ID auth service (Nest.js)

## Installation (for debug)

```bash
$ npm install

# dev mode
$ npm run start
/
# (hot reloading)
$ npm run start:dev
```

## Running the service

```bash
# building docker
$ docker-compose -f docker-compose.production.yml up --build
```
